mvp.ui

Functions

mvp.ui.DrawIcon(x, y, unicode, size, color, style)

 View source »

Draws Font Awesome icon.

Parameters

  • x number

    X position of the icon.

  • y number

    Y position of the icon.

  • unicode string

    Unicode of the icon from Font Awesome site.

  • size number default: 16

    Size of the icon.

  • color Color default: color_white

    Color of the icon.

  • style string optional

    Style of the icon. Function will try to determine style automatically if not specified.

Returns

  • IconData

    Icon table.

mvp.ui.GetIcon(unicode, size, style)
Internal

This is an internal function! You are able to use it, but you risk unintended side effects if used incorrectly.

View source »

Gets Font Awesome icon

Parameters

  • unicode string

    Unicode of the icon from Font Awesome site.

  • size number default: 16

    Size of the icon.

  • style string optional

    Style of the icon. Function will try to determine style automatically if not specified.

Returns

  • IconData

    Icon table.

Example Usage

local icon = mvp.ui.GetIcon("check", 32)

draw.SimpleText(icon.text, icon.font, 0, 0, Color(255, 255, 255, 255), TEXT_ALIGN_CENTER, TEXT_ALIGN_CENTER)

mvp.ui.Register(name, panel, basePanel)

 View source »

Registers a new panel, and make sure it's can be reloded using concommand. Uses vgui.Register internally.

Parameters

  • name string

    The name of the panel.

  • panel Panel

    The panel to register.

  • basePanel string

    The base panel to use.
