mvp.ui.gradients

Functions

mvp.ui.gradients.CreateHTMLGradient(id, w, h, rotation, cornerRadius, startColor, endColor, ignoreCache, manual)

 View source »

Creates a gradient material from HTML

Parameters

  • id string

    The id of the gradient for caching purposes

  • w number

    The width

  • h number

    The height

  • rotation string

    The rotation

  • cornerRadius string

    The corner radius

  • startColor string

    The start color

  • endColor string

    The end color

  • ignoreCache boolean

    Whether to ignore the cache

  • manual boolean

    Whether to return the HTML panel

Returns

  • string

    The id of the gradient

Example Usage

local gradient = mvp.ui.gradients.CreateHTMLGradient("test", 100, 100, "0deg", "0px", "rgba(32,32,32,1)", "rgba(51,51,51,1)")
mvp.ui.gradients.DrawHTMLGradient(gradient, 0, 0, 100, 100)
